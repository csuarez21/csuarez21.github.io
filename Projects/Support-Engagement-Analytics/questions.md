# Support & Engagement Analytics Project

This project analyzes user engagement and customer support performance using SQL Server.

## Completed Analyses

### 2. First Response Time (FRT)

### 3. SLA Compliance
Measured percentage of tickets meeting defined response and resolution time thresholds by priority and category.

### 4. Backlog Trend
Calculated open tickets at end-of-day across the reporting period.

### 5. Agent Productivity
Analyzed agent performance including tickets touched, reply volume, and average first response time.
